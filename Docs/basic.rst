**Basic usage description**
------------------------------


.. image:: 483026fbdc47496f8b140360f8bcbb0c/media/image1.png

   :height: 3.90833in
   
(1)  Allows scaling of the brain   
(2)  Global control of all structures, using the slider will render

     disappear
